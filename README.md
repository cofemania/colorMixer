# 🎨 ColorMixer – умный подбор смесей красок

**ColorMixer** – это инструмент для моделистов, художников и всех, кто работает с акриловыми красками (Vallejo, AK Interactive и другие). Программа автоматически подбирает смесь из имеющихся у вас красок, максимально точно соответствующую заданному цвету (HEX).

## ✨ Возможности

- **Автоматический подбор** – программа сама решает, смешивать 2 или 3 краски, а также пробует равные доли (1:1, 1:1:1) и все сочетания от 1/2 до 7/8 для двух красок.
- **Высокая точность** – используется цветовое пространство **CIELAB** и метрика **Delta E 2000** (наиболее соответствующая человеческому восприятию).
- **Молниеносная скорость** – благодаря **Numba JIT**, **Cython** (опционально) и **многопроцессорности** (все ядра CPU).
- **Поддержка дубликатов** – если один и тот же HEX-код встречается у нескольких производителей, программа покажет все альтернативы в скобках.
- **Простой формат входных данных** – CSV-файл с колонками `article`, `name`, `hex`, `brand`.
- **Интерактивный режим** – никаких аргументов командной строки, просто запустите скрипт и введите целевой HEX.

## 🖥️ Требования

- **Python 3.11 или новее** (рекомендуется 3.11–3.13)
- Операционная система: Windows / Linux / macOS
- Достаточно оперативной памяти для загрузки вашей таблицы красок (обычно хватает 1–2 ГБ)

## 📦 Установка

1. **Клонируйте репозиторий**

 ```
 git clone https://github.com/cofemania/colorMixer.git
 cd ColorMixer
 ```
   
2. **Установите зависимости**

```
pip install pandas numpy colormath2 tqdm numba
```
Для максимальной скорости можно также установить Cython и скомпилировать модуль (см. раздел «Оптимизация»).

3. **Запустите скрипт**

```
python ColorMixer.py
```

##⚙️ Оптимизация производительности
Программа автоматически использует Numba для ускорения вычислений Delta E. Если Numba не установлена, она работает через стандартную библиотеку colormath (медленнее).

Для ещё более высокой скорости (в 50–100 раз) можно скомпилировать Cython-модуль:

Установите Cython:
```
pip install cython
```

Запустите компиляцию:
```
python setup.py build_ext --inplace
```

Скопируйте полученный .pyd (Windows) или .so (Linux/macOS) в папку со скриптом.

Многопроцессорность включается автоматически – программа задействует все ядра CPU, кроме одного (чтобы система оставалась отзывчивой). При необходимости можно изменить число ядер в коде.

❓ Часто задаваемые вопросы
Q: Что делать, если скрипт не видит файл paints.csv?
A: Поместите его в ту же папку, что и скрипт, и проверьте название (регистр не важен, но расширение .csv обязательно).

Q: Почему некоторые смеси показывают совпадение 100%, хотя визуально цвет отличается?
A: Метрика Delta E 2000 очень точна, но на восприятие влияют также текстура, глянец и освещение. Для идеального результата рекомендуем проверять смесь на практике.

Q: Можно ли использовать больше 3 красок?
A: В текущей версии – нет, так как перебор 4+ красок экспоненциально сложен. При необходимости вы можете изменить код, но время работы возрастёт.

Q: Поддерживаются ли краски других брендов?
A: Да, любые, для которых вы укажете HEX-код в файле paints.csv.
